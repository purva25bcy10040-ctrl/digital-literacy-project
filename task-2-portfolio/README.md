 Task 2 – Student Digital Portfolio

 Objective
The objective of this task was to create a professional digital presence by setting up student profiles on important academic and technical platforms.

3. HackerRank
HackerRank is a coding practice platform used to improve programming and problem-solving skills through challenges and assessments.
